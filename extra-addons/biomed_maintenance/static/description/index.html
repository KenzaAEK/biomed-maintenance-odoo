BioMed Maintenance Intelligente

La solution ERP pour BioMed Tanger

Module développé pour le Projet PFE - Génie Logiciel

Ce module étend les capacités standards d'Odoo pour répondre aux exigences critiques du secteur biomédical. Il intègre une couche d'Intelligence Artificielle pour sécuriser les interventions techniques.

Fonctionnalités Clés

IA & Triage Automatique

Analyse sémantique (NLP) des descriptions de pannes pour détecter les urgences vitales.

Sécurité Biologique

Détection automatique des risques de contamination (Sang, Virus) et alertes EPI.

Intégration Totale

Lien natif avec les numéros de série (Stock) et les contrats de vente (Sales) de la Partie 1.
